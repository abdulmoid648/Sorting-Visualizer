 Overview
This console-based application allows users to input a custom list of numbers and watch as the program sorts them using one of four algorithms. Unlike standard array-based visualizers, this project focuses on **Linked List pointer manipulation**.

Algorithms Featured
Merge Sort: Demonstrates recursive "Divide and Conquer" logic ($O(n \log n)$).
Bubble Sort: Visualizes the "bubbling" of largest elements to the end.
Insertion Sort: Shows how elements are placed into a growing sorted sub-list.
Selection Sort: Highlights the process of finding the minimum element in each pass.

Technical Highlights
Dynamic Memory: Manual management of `Node` structures.
Fast/Slow Pointers: Used for finding midpoints in Merge Sort.
Input Validation: Robust error handling for non-numeric or out-of-bounds inputs.

 How to Run
1. Clone the repository.
2. Compile using any C++ compiler (e.g., `g++ main.cpp -o visualizer`).
3. Run the executable: `./visualizer`.# Sorting-Visualizer
